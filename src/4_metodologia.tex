\chapter[Metodologia]{Metodologia}
\label{cap:metodologia}

A metodologia para o desenvolvimento do trabalho proposto será dividida nas seguintes etapas.

\begin{enumerate}
  \item Levantamento do referencial teórico sobre arquiteturas de software e desenvolvimento de aplicações móveis multiplataforma.
  \item Leitura e análise de referenciais teóricos.
  \item Levantamento e verificação de pontos que influenciam a qualidade das arquiteturas de software.
  \item Descrever uma nova arquitetura de desenvolvimento de software que atenda as necessidades de desenvolvimento de uma aplicação móvel multiplataforma.
  \item Avaliação da arquitetura proposta.
\end{enumerate}
