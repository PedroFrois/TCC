\chapter[Conclusão]{Conclusão}
\label{cap:conclusao}

Nessa primeira parte do projeto foram realizadas todas as atividades previstas pelo cronograma conforme a Figura \ref{fig:cronograma}. Com o final das atividades propostas, é possível concluir que o andamento do trabalho conseguirá seguir o cronograma planejado, apesar de alguns pequenos atrasos ao longo do semestre. Ademais, conclui-se que o projeto de uma nova arquitetura voltada para o desenvolvimento de aplicações móveis multiplataforma é viável e provavelmente atingirá os objetivos esperados, uma vez que uma breve descrição já foi atingida ao final da matéria de TCC 1.
